
# 组件库快速成型工具

### 介绍

开箱即用的 `Vue3组件库` 快速成型工具，提供了一系列命令和工具去解决组件库开发上的问题。

### 特性
- 📦 &nbsp;开箱即用的组件库开发环境
- 📦 &nbsp;开箱即用的组件库编译工具，支持导出 `esm`, `cjs`, `umd` 三种模块代码
- 🛠️ &nbsp;支持 `sfc` 和 `tsx` 两种风格的组件库编写风格
- 📦 &nbsp;开箱即用的单元测试工具
- 💪 &nbsp;支持 `Typescript`
- 🚀 &nbsp;基于 `pnpm`
### 快速开始

`@xixi/cli` 内置了 `sfc` 和 `tsx` 两种风格的组件库项目模板，可以通过 `create` 命令直接生成。
方便您直接进入组件库开发。

```shell
# playground-ignore
pnpm add @xixi/cli -g
xixi-cli create
```
